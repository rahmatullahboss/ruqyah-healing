নিচে বদনজরের সম্ভাব্য লক্ষণসমূহ উল্লেখ করা হলো। তবে মনে রাখবেন — এই লক্ষণগুলোর বেশিরভাগই **অভিজ্ঞতার আলোকে প্রমাণিত**, ওহী থেকে নয়। সুতরাং ভুল হওয়ার সম্ভাবনাও রাখে।

---

## বদনজরের সাধারণ লক্ষণ

| ক্রমিক | লক্ষণ |
|--------|-------|
| ১ | ব্যবসা, চাকরি, আয়রোজগারে উন্নতি না হওয়া; সবকিছু অবনতির দিকে যাওয়া |
| ২ | পেশাগত কাজ করতে গেলেই ঝামেলা সৃষ্টি বা অসুস্থ হয়ে যাওয়া |
| ৩ | কাজকর্মে মনোযোগ না বসা |

| ৫ | ভুলে যাওয়ার প্রবণতা বেড়ে যাওয়া |
| ৬ | কাজকর্মে ঘনঘন ভুল করা |
| ৭ | স্বপ্নে লাল চোখওয়ালা বা বোরকা পরা (চোখ খোলা) মানুষ দেখা |
| ৮ | স্বপ্নে মরা মানুষ বা নিজেকে মৃত দেখা |

| ১৩ | একের পর এক বিভিন্ন ধরনের রোগ লেগে থাকা |
| ১৪ | অসুখ থাকা কিন্তু মেডিকেল টেস্টে কারণ না পাওয়া |
| ১৫ | সাধারণ রোগে দীর্ঘদিন আক্রান্ত থাকা, ওষুধে কাজ না হওয়া |
| ১৬ | শরীরে জ্বর থাকলেও থার্মোমিটারে না ওঠা |
| ১৭ | জ্বর জ্বর ভাব থাকা |
| ১৮ | বমি বমি ভাব |
| ১৯ | ঘুম ঘুম ভাব |
| ২০ | অতিরিক্ত হাই তোলা |
| ২১ | শরীর প্রায়ই দুর্বল থাকা |
| ২২ | হঠাৎ স্বাস্থ্য ভেঙে পড়া |
| ২৩ | দিনদিন রোগাটে হতে থাকা |
| ২৪ | চেহারায় লাল বা কালো ছোপ ছোপ দাগ |

| ২৮ | শরীরের বিভিন্ন জায়গায় গুটির মতো অনুভব |
| ২৯ | ব্রণ বা চর্মরোগের প্রকোপ বেড়ে যাওয়া |
| ৩০ | প্রচুর গ্যাস্ট্রিক, শক্তিশালী ওষুধেও সারছে না |

| ৩৪ | হাত-পায়ে ব্যথা বা পুরো শরীরে ব্যথা দৌড়ানো |
| ৩৫ | মাথা ঝিম ধরে থাকা |
| ৩৬ | কাঁধ, মাথার পেছনে বা উপরে ভার হয়ে থাকা |
