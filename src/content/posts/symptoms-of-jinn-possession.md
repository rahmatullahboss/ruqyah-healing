জ্বীনের আছরের লক্ষণগুলো দুইভাবে বিভক্ত:
১) নিদ্রাবস্থার লক্ষণ
২) জাগ্রত অবস্থার লক্ষণ

*('বিকায়াতুল ইনসান মিনাল জিন্নি ওয়াশ শায়তান' কিতাব থেকে অনুবাদিত)*

---

## জাগ্রত অবস্থার ২৪টি লক্ষণ

| ক্রমিক | লক্ষণ |
|--------|-------|
| ১ | চোখ, নাক, কান, দাঁত, গলা বা পাকস্থলীর কোনো সমস্যা ছাড়াই সর্বদা **মাথাব্যথা** — ওষুধে সারে না |
| ২ | যিকর, নামায ও অন্যান্য ইবাদাতের আগ্রহ নষ্ট হওয়া (**ইবাদাতবিমুখতা**) |
| ৩ | বিভিন্ন রকমের **মানসিক রোগের লক্ষণ** প্রকাশ পাওয়া |
| ৪ | প্রচণ্ড শারীরিক **দুর্বলতা ও অলসতা** |
| ৫ | **মৃগীরোগ** |
| ৬ | ডাক্তারি পরীক্ষায় কারণ না মিললেও শরীরের কোনো অঙ্গ **বিকল** হয়ে যাওয়া বা ব্যথা থাকা |
| ৭ | সর্বদা **মেজাজ বিগড়ে থাকা** |
| ৮ | **অকারণে কান্নাকাটি** করা |
| ৯ | যখন-তখন **রেগে যাওয়া** |
| ১০ | **অস্বাভাবিক আচরণ** করা |
| ১১ | ইচ্ছা না থাকলেও ব্রেনের নিয়ন্ত্রণ হারিয়ে **খারাপ আচরণ** প্রকাশ পাওয়া |
| ১২ | হঠাৎ **অজ্ঞান বা বেহুশ** হয়ে যাওয়া |
| ১৩ | **শ্বাসকষ্ট** শুরু হওয়া |
| ১৪ | দাঁতে-দাঁতে **খিল লেগে যাওয়া** |
| ১৫ | মুখ থেকে **ফেনা** বের হওয়া |
| ১৬ | চেহারার রঙ **ফ্যাকাশে** হয়ে যাওয়া |
| ১৭ | কেউ ডাকছে মনে হওয়া — অথচ **কাউকে খুঁজে না পাওয়া** |
| ১৮ | সবার মাঝে থেকেও **অদৃশ্য কণ্ঠস্বর** শুনতে পাওয়া |
| ১৯ | জাগ্রত অবস্থায় স্বপ্নের মতো বিষয় দেখা (**হ্যালুসিনেশন**) |
| ২০ | **একাকী ও নির্জনে থাকতে পছন্দ** করা |
| ২১ | সবসময় **ভীতু ভীতু ভাব** থাকা |
| ২২ | হঠাৎ করে **ভিন্ন ভাষায় বা ভিন্ন ভঙ্গিতে কথা** বলা |
| ২৩ | অন্য ধর্মাবলম্বীদের সাজ-সজ্জা পছন্দ করা বা **ভিন্ন ধর্মের রীতি-নীতি প্রকাশ** পাওয়া |
| ২৪ | সন্তানকে **বুকের দুধ না খেতে দেওয়া** |

---

> **বিঃদ্রঃ:** উপরের লক্ষণগুলোর মধ্যে একাধিক লক্ষণ লম্বা সময় ধরে প্রকাশ পেলে এবং মেডিকেল চিকিৎসায় সুফল না মিললে রুকইয়াহ ডায়াগনোসিস করানো উচিত।
