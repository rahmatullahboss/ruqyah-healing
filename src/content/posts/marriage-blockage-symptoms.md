বিবাহ হতে বিলম্ব হওয়া মানেই বিবাহ বন্ধ বা বান মারা নয় — তবে এটা বিবাহ না হওয়ার **অন্যতম একটি কারণ।**

---

## বিবাহ বন্ধের যাদুর ২২টি লক্ষণ

| ক্রমিক | লক্ষণ |
|--------|-------|
| ১ | বাহ্যিক কারণ ছাড়াই প্রায়ই সর্দি লেগে থাকা |
| ২ | মানসিক অস্থিরতা |
| ৩ | বুকের মাঝে সংকীর্ণ ভাব, বিশেষত আসর ও মাগরিবের পরে |
| ৪ | প্রায়ই পেটে ব্যথাভাব |
| ৫ | ঘুমের মধ্যে প্রায়ই ভয় পাওয়া |
| ৬ | ঘুমাতে না পারা |
| ৭ | ঘুমের মধ্যে অস্বস্তি বোধ |
| ৮ | ঘুম থেকে উঠেও দীর্ঘক্ষণ কষ্ট অনুভব |
| ৯ | যত সুন্দর প্রস্তাবই হোক, কোনোটাই মনমতো না হওয়া |
| ১০ | প্রস্তাবদাতাকে অযথাই সন্দেহ ও ঘৃণার চোখে দেখা |
| ১১ | অকারণেই খারাপ মনে হওয়া |
| ১২ | সর্বদা আজেবাজে চিন্তা মাথায় আসা |
| ১৩ | প্রায়শই মাথাব্যথা এবং ওষুধেও ভালো না হওয়া |
| ১৪ | সামান্যতেই রেগে যাওয়া |
| ১৫ | মেরুদণ্ডের নিচের দিকে হাড়ে ব্যথা |
| ১৬ | বিয়ের কথাবার্তা শুরু হলেই অসুস্থ হয়ে যাওয়া |
| ১৭ | কোনো সমস্যা না থাকলেও বিয়েতে আগ্রহ না থাকা |
| ১৮ | প্রস্তাব আসার আগে আগ্রহ থাকা, কিন্তু প্রস্তাব আসলেই আগ্রহ হারানো |
| ১৯ | বিয়ের কথাবার্তা শুরু হলেই চেহারা-স্বাস্থ্য নষ্ট হয়ে যাওয়া |
| ২০ | স্বপ্নে খাওয়া-দাওয়া করা |
| ২১ | স্বপ্নে শারীরিকভাবে নির্যাতিত হওয়া |
| ২২ | এগুলোর পাশাপাশি জ্বীন আছরের অন্যান্য লক্ষণ প্রকাশ পাওয়া |

---

> **বিঃদ্রঃ** উপরের ২/১টা লক্ষণ মাঝেমধ্যে দেখা দিলেই বান মারা হয়েছে বলে ধরে নেওয়া যাবে না। অধিকাংশ লক্ষণ প্রায়ই থাকলে এবং মেডিকেল টেস্টে কারণ না মিললে রুকইয়াহ ডায়াগনোসিস করানো উচিত।
